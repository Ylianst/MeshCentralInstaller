MeshCentral Installer
=====================

For more information, [visit MeshCommander.com/MeshCentral2](http://www.meshcommander.com/meshcentral2).

This is the Windows installer for MeshCentral. For expert users it's probably best to install MeshCentral using the command line with "npm install meshcentral", but this installer automates the steps of installing NodeJS and getting MeshCentral running.
